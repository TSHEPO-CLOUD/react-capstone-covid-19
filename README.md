### Prerequisites

A Morden Browser

### Setup
- Go to [https://github.com/TSHEPO-CLOUD/react-capstone-covid-19]
- Clone the project to your local machine
- Cd into your directory
- Run npm install
- Type: npm run build
- Run npm start

### Setup
- To Test npm install jest and run npm test
